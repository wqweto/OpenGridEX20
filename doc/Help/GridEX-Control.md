# GridEX Control

## GridEX Control

![](images/GridEX.jpg)  
 Provides an intuitive user interface for data manipulation.

### Syntax

 **GridEX**

### Remarks

 The Janus **GridEX** control gives an intuitive user interface for database applications.  
 With a **GridEX** control, you can sort and group related records, and you can choose to display the data in a Recordset as rows in a grid or as individual cards.  
 With the **GridEX** control, a user can drag a column header and drop it into the “group by” box and the control will automatically group and display the records using the values of the chosen field. When a user adds or updates a record, the **GridEX** control places this new record in the appropriate position based on the current sort and/or group settings.  
 Each cell of a **GridEX** control can hold text values and/or icons. The cell is displayed depending on its column settings. A column can be displayed as text, icon and text, icon and as a check box. In addition, a cell can be edited as a text box, check box, drop-down list, combobox or as a drop-down calendar.  
 You can use the **GridEX** control as a standard data grid or as a cardholder, where each card represents a record in a **Recordset**.  
 You can bind the **GridEX** to a myriad of databases using DAO 3.6 or ADO 2.x, or you can use the control in unbound mode to display custom data in any format.
