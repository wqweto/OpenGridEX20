# GEXPreview Control

## GEXPreview Control

![](images/GEXPreview.jpg)  
 Acts as a print preview window for a **GridEX** control

### Syntax

 **GEXPreview**

### Remarks

 You can use this ActiveX control to provide print preview funtionality to applications where **GridEX** control's content can be printed.  
 This control is used in conjunction with the **GridEX**'s **PrintPreview** method to provide you with a flexible and powerful way to implement a print preview window for **GridEX** control's content.  
 The control has a built-in toolbar to allow users to change the page layout, move between pages or select the zoom they want to see in the pages.
